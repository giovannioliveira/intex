4 API reference
\newclass{<name>}{<pretty name>}{<listof title>} initializes a new class
envirionment. <name> is used to name the environment and as preﬁx or
suﬁx to compose its command names. <pretty name> is used in captions.
<listof title> is used as title for the list index.
\include<name>{<width>}{<height>}{<metadata>} includes an empty place-
holder with given dimentions and metadata.
\includeg<name>[<opt>]{<img>}{<metadata>} includes an image placeholder
using <img> with given metadata. <opt> is passed as option to \includegraphics.
\<name>frame{<swidth>}{<sheight>}{<metadata>} calls \include<name> in-
side a centered frame with given arguments. The values <swidth> and
<sheight> are scaled to \textwidth.
\<name>framecap{<swidth>}{<sheight>}{<metadata>}{<caption>} the same
as \<name>frame, but with caption.
\<name>gframe[<opt>]{<img>}{<metadata>} calls \includeg<name> inside a
centered frame with given arguments.
\<name>gframe[<opt>]{<img>}{<metadata>}{<caption>} the same as \<name>gframe,
but with caption.
\listof<name> make the list of environment using default style.
